An extension trait for easily formatting missing values.

## Example

```rust
use fmtor::FmtOr;

let maybe_box: Option<Box<()>> = None;

println!("The box is at: {:p}", maybe_box.fmt_or("Null"));
```

Prints: *The box is at: Null*
